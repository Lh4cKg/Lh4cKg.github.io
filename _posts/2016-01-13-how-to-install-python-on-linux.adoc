= როგორ დავაყენოთ პითონი ლინუქსზე
:hp-alt-title: how to install python on linux

მარტივი გზა პითონის დასაყენებლად არის package manager ის გამოყენება, როგორებიცაა apt-get, yum, pacman და ა.შ.
